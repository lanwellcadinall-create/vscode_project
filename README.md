# Тесты для API Яндекс.Самокат

## Описание проекта
Автотесты для проверки API сервиса "Яндекс.Самокат"

## Проверяемый сценарий
1. Создание заказа (POST /api/v1/orders)
2. Получение номера трека заказа
3. Получение данных о заказе по треку (GET /api/v1/orders/track?t={track})
4. Проверка кода ответа 200

## Документация API
Документация доступна после запуска стенда по адресу:
`https://db12b8e4-023a-4821-9bfe-3590adfbd328.serverhub.praktikum-services.ru/`
